# React micro frontends with Webpack module federations

Monorepo with React project to explain how Module federations work's.

## Install and running

Run `yarn` on root folder. Run the app2 first, athen, run the app1.

### Running app 01

`yarn start:app1`

### Running app 02

`yarn start:app2`

### Building app 01

`yarn build:app1`

### Building app 02

`yarn build:app2`
